Key Features

  • 3-state bottom sheet: collapsed pill, peek with last message, full chat
  • Inline product carousels with direct Add to Cart
  • Context-aware suggestions based on current browsing
  • Cart awareness with running total displayed
  • Proactive nudges: "I see you're browsing Indica..."
  • Suggestion chips for quick tap interactions

Advantages

  • Familiar mobile pattern (iMessage, Maps)
  • Non-intrusive — user controls engagement level
  • Natural conversational flow
  • Low learning curve for users

Challenges

  • Limited screen real estate when expanded
  • Requires typing on mobile (friction)
  • Chat history can get long and hard to navigate

Challenges

  • High implementation complexity (content generation)

Recommendation

The Hybrid Approach

No single strategy serves all users optimally. We recommend a phased hybrid approach that combines the best elements of each strategy, starting with the lowest-friction, highest-impact foundation and layering on sophistication over time.

Phase 1 — Foundation
Bottom Sheet Budtender
Launch with Strategy 1 as the core interaction layer. It is the most familiar pattern, lowest implementation risk, and works across all devices. This becomes the primary AI touchpoint.
Phase 2 — Onboarding
Guided Journey for New Users
Add Strategy 4's visual onboarding quiz for first-time visitors. Use responses to personalize the bottom sheet's suggestions and pre-populate preferences. Skip for returning users.
Phase 3 — Intelligence
Copilot Insights in Chat
Integrate Strategy 3's proactive insights directly into the bottom sheet. Product comparisons, educational nuggets, and bundle suggestions appear as chat cards rather than a separate sidebar.
